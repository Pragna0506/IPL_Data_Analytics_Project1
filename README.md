# 🏏 IPL Data Analytics Dashboard

An interactive data analytics dashboard developed during a Data Analytics workshop to demonstrate data visualization techniques using Python and Streamlit. The project focuses on exploring IPL data through interactive charts, graphs, and insights to transform raw data into meaningful visual stories.

🚀 Features

- 📊 Interactive IPL Data Dashboard
- 📈 Data Visualization using Plotly
- 🏏 Team and Player Performance Analysis
- 📉 Statistical Insights and Trends
- 🎨 Interactive Charts and Graphs
- 🌐 Streamlit Web Application

🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Plotly
- NumPy

📊 Dataset

- The IPL dataset was used to perform exploratory data analysis and create interactive visualizations for understanding team performances, match statistics, and overall tournament trends.

🌐 Live Demo

The application is deployed on Streamlit Cloud.

🔗 Live Application: https://pragna0506-ipl-data-analytics-project1-app-mmyezg.streamlit.app/

🎯 Learning Outcomes

- Data Cleaning and Preparation
- Exploratory Data Analysis (EDA)
- Data Visualization Techniques
- Dashboard Development with Streamlit
- Deployment using Streamlit Cloud
